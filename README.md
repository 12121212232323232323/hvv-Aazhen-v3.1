# ThinkphpGUI
自主开发的第一个JAVA图形化漏扫工具，支持ThinkPHP 2.x RCE，Thinkphp5 5.0.22/5.1.29RCE，ThinkPHP5 5.0.23RCE和ThinkPHP5 SQL注入漏洞和敏感信息泄露漏洞。
因为刚学java没多久，所以工具做得不是很好，功能不多，非常简陋，会一直更新持续改进，这辈子不会放弃java GUI，直至海枯石烂！
